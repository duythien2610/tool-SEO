## ✨ Tổng Quan

**Digital Marketing Workbench** là web tool client-side hoàn toàn, được xây dựng để mô phỏng quy trình làm việc thực tế của một **Senior Digital Marketing & SEO Executive** trong môi trường doanh nghiệp.

> 🎯 **Mục tiêu:** Demo năng lực vận hành Digital Marketing end-to-end — từ SEO Content đến Analytics, CRO, và Reporting — trong một giao diện tích hợp duy nhất.

---

## 🗂️ Tính Năng (8 Modules)

| Module | Tính Năng | Mô Tả |
|--------|-----------|-------|
| 📊 **Dashboard** | KPI Calculator | Tự động tính CTR, CVR, CPL, ROAS, CPC từ dữ liệu nhập |
| | Skill Matrix | Bản đồ kỹ năng Digital Marketing theo JD chuẩn |
| ✍️ **SEO** | Content Brief Generator | Tạo brief + AI writing prompt chuẩn GEO/AI Search |
| | Audit Checklist | 17 điểm kiểm tra kỹ thuật SEO + GEO/AI Overviews |
| 🔑 **Keywords** | Intent Mapper | Phân loại keyword: Informational / Commercial / Transactional / Local |
| 📣 **Channels** | SEM Planner | Lập cấu trúc campaign Google Ads / Meta / LinkedIn / TikTok |
| | Social Calendar | Lịch nội dung 4 tuần theo platform |
| | Email Sequence | Chuỗi email tự động: Welcome / Lead nurture / Re-engagement |
| 📈 **Analytics** | UTM Builder | Build link tracking chuẩn naming convention |
| | Tracking Plan | Kế hoạch Event tracking cho GTM / GA4 |
| 🧪 **CRO Tests** | Experiment Backlog | Quản lý A/B test với điểm ICE (Impact · Confidence · Ease) |
| ⚙️ **Ops** | Rank Tracker | Theo dõi thứ hạng từ khóa theo tuần |
| | GBP Checklist | Quản lý Google Business Profile chuỗi chi nhánh |
| | Vertical Cards | Theo dõi task đa phòng ban với cảnh báo Overdue |
| 📄 **Reports** | Weekly Report Generator | Tự động tổng hợp báo cáo từ dữ liệu workspace |

👉 **[https://duythien2610.github.io/tool-SEO/](https://duythien2610.github.io/tool-SEO/)**

### Chạy Locally
```bash
git clone https://github.com/duythien2610/tool-SEO.git
cd tool-SEO
# Mở index.html trong trình duyệt
start index.html   # Windows
open index.html    # macOS
```
